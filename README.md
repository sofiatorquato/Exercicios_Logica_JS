# 🚀 Exercícios de Lógica de Programação

Este repositório contém resoluções de exercícios fundamentais para consolidar o aprendizado em **JavaScript**, focando em lógica, sintaxe e estruturas básicas.

---

## 📂 Organização dos Exercícios

### 1. Lógica Básica
- **Soma Simples**: Cálculo de dois números.
- **Conversor de Temperatura**: Conversão de Celsius para Fahrenheit.
- **Cálculo de Área**: Área de um retângulo.
- **Média do Aluno**: Média aritmética de notas.
- **Troca de Valores**: Inversão de valores entre variáveis.

### 2. Estruturas Condicionais
- **Par ou Ímpar**: Verificação via resto.
- **Maior de Idade**: Validação de permissão.
- **Calculadora de IMC**: Classificação de peso.
- **Aprovação Escolar**: Verificação de status.
- **Dia da Semana**: Uso de `switch`.

### 3. Estruturas de Repetição
- **Contagem Regressiva**: Laço `for` reverso.
- **Tabuada**: Geração dinâmica de tabuada.
- **Soma de 1 a 100**: Acumulação com `for`.
- **Lista de Nomes**: Manipulação de `Arrays`.
- **Fatorial**: Cálculo matemático recursivo/repetitivo.
